1. Create a new project folder

2. Create a virtual environment 
    py -m venv venv 

3. Activate venv
    venv\Scripts\activate

4. Install django
    pip Install django

5. Startproject
django-admin startproject core
py manage.py startapp blog
Go to core/settings.py

Installed apps -> add 'blog'

6. Create a repo
On terminal 
git init
